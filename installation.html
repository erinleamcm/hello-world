Installation

Once the ZIP file downloads, go to your download folder and double click on it to expand and be able to see all included files. There is a starting page waiting for your customization! Add the html provided at the bottom of this site's example pages to include your desired elements.

To further customize your design, go into the downloaded css folder and open filemanagement.css. Changing the 1rem value in line 364 "--text-size-normal: 1rem;" will change text sizes throughout the site. Lines 424, 434, 443, and 451 control the font-family choices. If you would like more customization control, use a preprocessor and the scss folder files to make as many changes to the template as you need. Have fun!
